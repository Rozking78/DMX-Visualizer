# Gobo Texture Assets List

This document lists all gobo textures needed for the DMX Shape Visualizer. Each gobo should be a **256x256 PNG** with:
- **White** = light passes through (opaque areas)
- **Black** = light blocked (transparent areas)
- **Grayscale** = partial light (feathered edges)

Place all gobo files in: `Resources/gobos/` or `gobos/` next to the executable.

---

## Naming Convention

```
gobo_[ID]_[name].png
```

Examples:
- `gobo_051_leaves_sparse.png`
- `gobo_111_galaxy_spiral.png`

## Technical Specifications

### Image Requirements
- **Format:** PNG with alpha channel
- **Size:** 256x256 pixels (will be scaled by app)
- **Color Mode:** Grayscale or RGB
- **Transparency:** Use alpha for complex edges

### Color Mapping
The app uses gobo images as **masks**:
- **White (255)** = Full light (DMX color shows through)
- **Black (0)** = No light (blocked)
- **Gray (1-254)** = Partial light (for soft edges)

### File Organization
```
GeoDrawNDI.app/
└── Contents/
    └── Resources/
        └── gobos/
            ├── gobo_051_leaves_sparse.png
            ├── gobo_052_leaves_dense.png
            └── ... (all 150 gobos)
```

Or next to executable:
```
gobos/
├── gobo_051_leaves_sparse.png
├── gobo_052_leaves_dense.png
└── ...
```

## Placeholder Generation

The app auto-generates procedural placeholders if PNG files are missing:
- **Breakups:** Random organic circles
- **Geometric:** Checkerboard pattern
- **Nature:** Wavy lines
- **Cosmic:** Star dots
- **Textures:** Noise rectangles
- **Architectural:** Grid lines
- **Abstract:** Radial gradient
- **Special:** "C1", "C2", etc. labels
