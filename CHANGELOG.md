# Changelog

All notable changes to DMX Visualizer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.3.0] - 2026-01-10

### Added
- Multi-mode DMX support (33ch Full, 23ch Standard, 10ch Compact)
- Console-style patch system with mode selection
- Layout Editor (Grid, Perimeter, Line, Rows layouts)
- MA3 Lua plugin for layout control
- 150 custom gobo textures
- Syphon and NDI output support
- In-app help system
- User Guide documentation

### Changed
- Redesigned Settings > DMX tab as Patch interface
- Improved fixture addressing for multi-universe support

### Technical
- Swift 6.2 + Metal rendering engine
- Apple Silicon optimized (M1/M2/M3/M4)
- Art-Net and sACN protocol support
